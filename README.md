# ISJO AI - Asistente de Voz Inteligente

Asistente inteligente con IA disponible 24/7 para automatizar la atencion telefonica de tu negocio.

## Tecnologias

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo local

```bash
pnpm install
pnpm dev
```

## Produccion

```bash
pnpm build
pnpm start
```

## Docker

```bash
docker build -t isjo-ai .
docker run -p 3000:3000 isjo-ai
```
